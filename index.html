wc-input

This extends the standard input to do some interesting things

Basic input

spans

Span sets the control to a preset width

Input span="narrow"
Input span="normal"
Intput span="wide"
Input span="extrawide"

pattern

Like input, pattern sets the allowed regex input but adds a few presets

Input pattern="\d+"
Input pattern="int"
Input pattern="+int"
Input pattern="-int"
Input pattern="float"
Input pattern="+float"
Input pattern="-float"

minmax

Min and max are supported as expected

Input min="10"
Input max="10"

fix

Fix rounds values to a set decimal place. It will not override what the user entered

Input fix="4"
Input fix="3"
Input fix="2"
Input fix="1"
Input fix="0"
Input fix="-1"
Input fix="-2"
Input fix="-3"
